Objects

Object is a non-primitive data type in JavaScript. It is like any other variable, the only difference is that an object holds multiple values in terms of properties and methods. Properties can hold values of primitive data types and methods are functions.

In other programming languages like Java or C#, you need a class to create an object of it. In JavaScript, an object is a standalone entity because there is no class in JavaScript.

Defining an object

var info = {
"name": "Oliver",
"age": 25
};

Retrieve value from object by key

var myObj = {"message": "Hello, earthling! I bring peace."};

console.log(myObj.message) // dot notation
console.log(myObj["message"]) // bracket notation

Add a property

var stackOverflow = {};

stackOverflow.isAllowed = true;

Remove a property

var thisSong = {"description": "The best song in the world."}

delete thisSong.description;
