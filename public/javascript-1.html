JavaScript Variables

Variable means anything that can vary. JavaScript includes variables which hold the data value and it can be changed anytime.

JavaScript uses reserved keyword var to declare a variable. A variable must have a unique name. You can assign a value to a variable using equal to (=) operator when you declare it or before using it.

Variable declaration and initialization

var firstName = "Alin"; // variable stores string value
var lastName = "Plamadeala";

Display in console

console.log("My first name is", firstName, "and my last name is", lastName);
console.log(`My first name is ${firstName} and my last name is ${lastName}`);
//Output: My first name is Alin and my last name is Plamadeala

JavaScript Data Types

JavaScript includes data types similar to other programming languages like Java or C#. Data type indicates characteristics of data. It tells the compiler whether the data value is numeric, alphabetic, date etc., so that it can perform the appropriate operation.

JavaScript includes primitive and non-primitive data types as per latest ECMAScript 5.1.

Primitive Data Types

  1. String
  2. Number
  3. Boolean
  4. Null
  5. Undefined

Non-primitive Data Type

  1. Object
  2. Date
  3. Array

Examples

var year = "2017"; //String literal
var number = 1; //Number 

// Add the year plus the number
var newYear = parseInt(year) + number; //or
var newYear = Number(year) + number; //or
var newYear = +(year) + 1;

console.log(newYear);
//Output: 2018

// Getting a character by index

var letters = "abc";
// Getting the character "c"

console.log(letters[2]);

console.log(letters.charAt(2));

console.log(letters.substring(2));


var fact = "You are learning javascript!";

// capitalize the J in Javascript

var newFact = fact.replace("j", "J");

console.log(newFact);

console.log(fact.indexOf("j"));
