First application

Creating an API is extremely simple with the help of Node.js and Express

Step 1

Navigate to your desired directory in the console using cd.

Step 2

Create a directory for your application using mkdir and cd into it.

mkdir first-api
cd first-api

Step 3

Create a file called package.json in this directory and paste the following code in it.

{
    "dependencies": {
        "express": "4.17.1"
    }
}

This will create a dependency for Express in your project.

Step 4

Install the dependencies via npm

npm install

Step 5

First, create a file called app.js and import the Express framework:

const express = require('express');

Now, let's create the Express app:

const app = express();

And set our port:

const port = 3000;

Now, we can create a simple GET endpoint. When a user hits the endpoint, the message "Hello World, from express" will appear. We'd like to set it to be on the home page, so the URL for the endpoint is /:

app.get('/', (req, res) => {
                res.send('Hello World, from express');
            });

At this point, let's start our clients:

app.listen(port, () => console.log(`Hello world app listening on port ${port}!`))

Step 6

Run the application and visit the only endpoint we have via our browser
